* Rofi Configuration with Tokyonight Theme

** Overview
   This configuration provides beautiful Rofi theming using Tokyonight-inspired designs with seamless switching between Rouge and Kanagawa variants.

** Directory Structure
   #+BEGIN_SRC text
   ~/.config/
   ├── rofi/
   │   ├── config.rasi              # Main configuration
   │   └── theme.rasi               # Active theme (auto-updated)
   └── themes/
       ├── rouge/
       │   └── rofi/
       │       └── theme.rasi       # Rouge variant
       └── kanagawa/
           └── rofi/
               └── theme.rasi       # Kanagawa variant
   #+END_SRC

** Main Rofi Configuration

   Save as =~/.config/rofi/config.rasi=:

   #+BEGIN_SRC css
   /* Rofi Configuration */
   @theme "theme"

   configuration {
       modi: "drun,run,window";
       show-icons: true;
       terminal: "alacritty";
       drun-display-format: "{icon} {name}";
       window-format: "{w} {i} {t}";
   }

   /* Window configuration */
   window {
       transparency: "real";
       width: 40%;
       location: center;
       anchor: center;
       y-offset: 0;
       x-offset: 0;
   }

   /* Main box */
   mainbox {
       children: [inputbar, listview, message];
   }

   /* Input bar */
   inputbar {
       children: [prompt, entry];
   }

   prompt {
       padding: 8px;
   }

   entry {
       padding: 8px;
   }

   /* Listview */
   listview {
       scrollbar: false;
       fixed-height: true;
       dynamic: true;
   }

   /* Elements */
   element {
       padding: 8px;
   }

   element-icon {
       size: 24px;
   }

   element selected {
   }
   #+END_SRC

** Theme Switcher Integration

*** Update Theme Switcher

    Add to =~/.local/bin/theme-switcher=:

    #+BEGIN_SRC bash
    # Apply rofi theme
    if [ -f "$theme_dir/rofi/theme.rasi" ]; then
        mkdir -p ~/.config/rofi
        cp "$theme_dir/rofi/theme.rasi" ~/.config/rofi/theme.rasi
        echo "  → rofi theme applied"
    fi
    #+END_SRC

** Installation

*** 1. Create Directories
    #+BEGIN_SRC bash
    mkdir -p ~/.config/rofi
    mkdir -p ~/.config/themes/{rouge,kanagawa}/rofi
    #+END_SRC

*** 2. Copy Configuration Files
    - Main config: =~/.config/rofi/config.rasi=
    - Rouge theme: =~/.config/themes/rouge/rofi/theme.rasi=
    - Kanagawa theme: =~/.config/themes/kanagawa/rofi/theme.rasi=

*** 3. Set Initial Theme
    #+BEGIN_SRC bash
    theme-switcher rouge
    #+END_SRC

*** Kanagawa Theme (Tokyonight Moon)

** Features

*** Both Themes Include
    - 40% width centered window
    - Rounded corners (8px border radius)
    - Icon support (24px size)
    - Real transparency
    - Three modes: drun, run, window
    - Proper focus states
    - Error message styling
    - Alacritty as default terminal

*** Color Systems
    - Consistent dark background (#1f1f28)
    - Parchment text colors (#dcd7ba)
    - Theme-specific accent colors
    - Proper contrast ratios
    - Cohesive color hierarchies

** Usage

*** Theme Switching
    #+BEGIN_SRC bash
    theme-switcher rouge      # Switch to Rouge theme
    theme-switcher kanagawa   # Switch to Kanagawa theme
    theme-switcher next       # Cycle between themes
    #+END_SRC

*** Manual Testing
    #+BEGIN_SRC bash
    # Test Rouge theme
    rofi -show drun -theme ~/.config/themes/rouge/rofi/theme.rasi

    # Test Kanagawa theme
    rofi -show drun -theme ~/.config/themes/kanagawa/rofi/theme.rasi

    # Use active theme
    rofi -show drun
    #+END_SRC

*** Rofi Modes
    - =rofi -show drun= :: Application launcher
    - =rofi -show run= :: Command runner
    - =rofi -show window= :: Window switcher

** Troubleshooting

*** Theme Not Applying
    - Verify =~/.config/rofi/theme.rasi= exists
    - Check main config references theme correctly
    - Test with explicit theme path

*** Icons Missing
    - Ensure =show-icons: true= in config
    - Verify system icon theme is installed
    - Check icon size settings

*** Transparency Issues
    - Ensure compositor (picom) is running
    - Change =transparency: "real"= to ="none"= if needed
