* Polybar Status Bar Configuration

  Complete Polybar configuration with dot workspace indicators and theme support.

** Configuration File
   #+BEGIN_SRC ini
   ;==========================================================
   ;
   ; POLYBAR CONFIGURATION
   ;
   ;==========================================================

   ; Include theme colors
   include-file = ~/.config/polybar/colors

   [bar/main]
   ; Monitor settings
   monitor = ${env:MONITOR:eDP-1}

   ; Dimensions
   width = 100%
   height = 24pt
   radius = 6.0
   fixed-center = true

   ; Colors
   background = ${colors.background}
   foreground = ${colors.foreground}

   ; Borders
   border-size = 0
   border-color = #00000000

   ; Padding and spacing
   padding-left = 0
   padding-right = 2
   module-margin-left = 1
   module-margin-right = 1

   ; Separators
   separator = |
   separator-foreground = ${colors.disabled}

   ; Fonts
   font-0 = "Fira Mono Nerd Font:size=10;3"
   font-1 = "Font Awesome 6 Free:style=Solid:size=10;3"

   ; Modules
   modules-left = i3 xwindow
   modules-center = date
   modules-right = filesystem pulseaudio memory cpu

   ; Behavior
   wm-restack = i3
   cursor-click = pointer
   cursor-scroll = ns-resize
   enable-ipc = true

   [module/i3]
   type = internal/i3
   format = <label-state> <label-mode>
   index-sort = true
   wrapping-scroll = false

   ; Dot indicators for workspaces
   label-focused = ●
   label-focused-background = ${colors.primary}
   label-focused-foreground = ${colors.background}
   label-focused-padding = 2

   label-unfocused = ○
   label-unfocused-background = ${colors.background-alt}
   label-unfocused-foreground = ${colors.disabled}
   label-unfocused-padding = 2

   label-visible = ◐
   label-visible-background = ${colors.secondary}
   label-visible-foreground = ${colors.background}
   label-visible-padding = 2

   label-urgent = ◓
   label-urgent-background = ${colors.alert}
   label-urgent-foreground = ${colors.background}
   label-urgent-padding = 2

   [module/xwindow]
   type = internal/xwindow
   label = %title:0:60:...%
   label-foreground = ${colors.foreground}

   [module/filesystem]
   type = internal/fs
   interval = 30
   mount-0 = /

   format-mounted = <label-mounted>
   format-mounted-prefix = " "
   format-mounted-prefix-foreground = ${colors.primary}

   label-mounted = %percentage_used%%

   format-unmounted = <label-unmounted>
   label-unmounted = %mountpoint% not mounted
   label-unmounted-foreground = ${colors.disabled}

   [module/pulseaudio]
   type = internal/pulseaudio
   format-volume = <label-volume> <bar-volume>
   label-volume = %percentage%%
   label-volume-foreground = ${colors.foreground}

   format-volume-prefix = " "
   format-volume-prefix-foreground = ${colors.primary}

   format-muted = <label-muted>
   format-muted-prefix = " "
   format-muted-prefix-foreground = ${colors.disabled}
   label-muted = muted
   label-muted-foreground = ${colors.disabled}

   bar-volume-width = 6
   bar-volume-foreground-0 = #98C379
   bar-volume-foreground-1 = #98C379
   bar-volume-foreground-2 = #98C379
   bar-volume-foreground-3 = #E5C07B
   bar-volume-foreground-4 = #E5C07B
   bar-volume-foreground-5 = #E06C75
   bar-volume-gradient = false
   bar-volume-indicator = |
   bar-volume-fill = ─
   bar-volume-empty = ─
   bar-volume-empty-foreground = ${colors.disabled}

   [module/memory]
   type = internal/memory
   interval = 2
   format-prefix = " "
   format-prefix-foreground = ${colors.primary}
   label = %percentage_used:2%%

   [module/cpu]
   type = internal/cpu
   interval = 2
   format-prefix = " "
   format-prefix-foreground = ${colors.primary}
   label = %percentage:2%%

   [module/date]
   type = internal/date
   interval = 1.0

   date = %a %b %d %I:%M %p
   date-alt = %Y-%m-%d %H:%M:%S

   label = %date%
   label-foreground = ${colors.primary}

   [settings]
   screenchange-reload = true
   pseudo-transparency = false
   #+END_SRC

** Module Descriptions

*** Modules Left
    - *i3*: Workspace switcher with dot indicators
    - *xwindow*: Current window title

*** Modules Center
    - *date*: Current date and time

*** Modules Right
    - *filesystem*: Disk usage with icon
    - *pulseaudio*: Volume control with visual bar
    - *memory*: RAM usage with icon
    - *cpu*: CPU usage with icon

** Dot Indicator System

*** Visual States
    | Dot | Color          | State    | Meaning                    |
    |-----+----------------+----------+----------------------------|
    | ●   | Primary color  | Focused  | Active workspace           |
    | ○   | Disabled color | Unfocused| Inactive workspace         |
    | ◐   | Secondary color| Visible  | Visible on other monitor   |
    | ◓   | Alert color    | Urgent   | Requires attention         |

*** Alternative Dot Styles
    #+BEGIN_SRC ini
    ; Option 1: Minimal dots
    label-focused = •
    label-unfocused = ·
    
    ; Option 2: Numbered dots
    label-focused = %index%:●
    label-unfocused = %index%:○
    
    ; Option 3: Geometric shapes
    label-focused = ■
    label-unfocused = □
    label-visible = ◧
    label-urgent = ⧄
    #+END_SRC

** Color Reference

*** Theme Colors (via included file)
    | Color Variable   | Usage                          |
    |------------------+--------------------------------|
    | background       | Main bar background           |
    | foreground       | Primary text color            |
    | primary          | Focused workspaces, accents   |
    | secondary        | Visible workspaces            |
    | alert            | Urgent workspaces, warnings   |
    | disabled         | Unfocused workspaces, muted   |
    | background-alt   | Alternative background        |

** Installation

   Save as =~/.config/polybar/config.ini=:

   #+BEGIN_SRC bash
   # Create directory if it doesn't exist
   mkdir -p ~/.config/polybar

   # Copy the configuration
   cp polybar-config.txt ~/.config/polybar/config.ini

   # Start polybar
   pkill polybar
   polybar main &
   #+END_SRC

** Usage Tips

*** Testing Configuration
    #+BEGIN_SRC bash
    # Test config syntax
    polybar main -c ~/.config/polybar/config.ini -t

    # Run with info logging
    polybar main -l info

    # Kill and restart
    pkill polybar
    polybar main &
    #+END_SRC

*** Module Debugging
    - Check =polybar -l info= for module errors
    - Verify font icons are installed
    - Ensure i3 is running for workspace module
