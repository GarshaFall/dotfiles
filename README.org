* Fedora i3 Desktop Configuration

  A minimal, efficient Fedora i3 desktop configuration with confirmed working Emacs keybindings and 4-workspace workflow.

** üöÄ Quick Start

*** Prerequisites
    - Fedora i3 Spin (pre-installed with i3)
    - Basic familiarity with terminal commands

*** Package Installation
    #+BEGIN_SRC bash
    # Update system
    sudo dnf update

    # Install essential packages
    sudo dnf install \
        xterm nemo firefox vlc feh micro emacs \
        gcc gcc-c++ clang python3 python3-pip \
        polybar rofi picom papirus-icon-theme \
        htop neofetch arandr unzip lxappearance \
        fontawesome-fonts fira-code-fonts \
        flameshot xclip xsel \
        network-manager-applet blueman pavucontrol \
        dunst nitrogen git ripgrep fd-find fzf

    # Install media codecs
    sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm
    sudo dnf install https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm
    sudo dnf install gstreamer1-plugins-{bad-*,good-*,base} gstreamer1-plugin-openh264 gstreamer1-libav --exclude=gstreamer1-plugins-bad-free-devel
    #+END_SRC

*** Font Setup
    #+BEGIN_SRC bash
    # Install Nerd Fonts for icons
    mkdir -p ~/.local/share/fonts
    cd ~/.local/share/fonts
    wget https://github.com/ryanoasis/nerd-fonts/releases/download/v3.1.1/FiraCode.zip
    unzip FiraCode.zip
    fc-cache -fv
    #+END_SRC

*** Emacs Navigation
    | Keybinding | Action          |
    |------------+-----------------|
    | Ctrl + F   | Focus right     |
    | Ctrl + B   | Focus left      |
    | Ctrl + N   | Focus down      |
    | Ctrl + P   | Focus up        |

*** Application Launchers
    | Keybinding       | Action                  |
    |------------------+-------------------------|
    | Ctrl + X Ctrl + T | Terminal (XTerm)       |
    | Ctrl + X Ctrl + F | File Manager (Nemo)    |
    | Ctrl + X Ctrl + B | Web Browser (Firefox)  |
    | Ctrl + X Ctrl + E | Editor (Emacs)         |
    | Ctrl + X Ctrl + M | Micro Editor           |
    | Ctrl + X Ctrl + D | Rofi Application Menu  |

*** Window Management Mode
    - Press =Ctrl + X /= to enter window mode
    - Then use: =0= (close), =1= (fullscreen), =2= (split vertical), =3= (split horizontal), =O= (focus next), =K= (kill)
    - Press =Ctrl + G= to exit mode

*** Traditional i3
    | Keybinding        | Action                    |
    |-------------------+---------------------------|
    | Super + Enter     | Terminal                  |
    | Super + D         | Rofi                      |
    | Super + 1-4       | Switch workspace          |
    | Super + Shift + 1-4 | Move window to workspace |

*** File Locations
    - =~/.config/i3/config= : Main i3 configuration
    - =~/.config/polybar/config.ini= : Status bar configuration
    - =~/.config/rofi/config.rasi= : Application launcher
    - =~/.Xresources= : Terminal configuration

*** Quick Setup
    #+BEGIN_SRC bash
    # Create directories
    mkdir -p ~/.config/{i3,polybar,rofi}

    # Copy configuration files to their locations
    # Then reload:
    i3-msg reload
    pkill polybar && polybar main &
    #+END_SRC

    - Some applications may capture =Ctrl+X= prefix
    - Use =Super= key as backup for critical functions
    - Test keybindings in clean i3 session

** üí° Workflow Tips

*** Efficient Navigation
    - Use =Ctrl+F/B/N/P= for quick window focus
    - Master =Ctrl+X /= for window management mode
    - Use workspace numbers (1-4) for instant context switching
    - Assign frequent apps to muscle memory keybindings

** üéØ Customization

*** Adding Applications
    #+BEGIN_SRC bash
    # Install new application
    sudo dnf install application-name

    # Add to i3 config
    bindsym $mod+key exec application-name

    # Or use Rofi (Super+D) to launch
    #+END_SRC

*** Modifying Workspaces
    - Edit workspace assignments in i3 config
    - Change auto-assignment rules
    - Adjust polybar dot indicators

*** Creating New Keybindings
    #+BEGIN_SRC ini
    # Add to ~/.config/i3/config
    bindsym $mod+key exec application
    bindsym Control+x key exec application
    #+END_SRC

** ‚ö° Performance Tips

*** System Optimization
    - Use =htop= to monitor resource usage
    - Disable unused services: =systemctl disable service-name=
    - Prefer terminal applications for lightweight operation
    - Use =ripgrep= instead of =grep= for faster searching

*** i3 Performance
    - Limit startup applications
    - Use simple wallpapers (Feh is lightweight)
    - Keep polybar modules minimal
    - Monitor with =i3-msg -t get_tree=
